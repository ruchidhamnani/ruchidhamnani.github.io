Bookstore Backend with MERN Stack

Database Management Systems

Domain: Database Management Systems, Software Engineering, Web Development

  • As a part of the project, I developed RESTful APIs following MVC Architecture in Java Script using Express framework.
  • We modelled the database for the system using Entity-Relationship Model and designed the relational database schema using the ER-diagram.
  • Also created a data schema for storing all the relevant data and respective data models required to serve all functionalities of a bookstore
  • The software was divided into two broad categories: admin and user. For the user, we included functionalities such as renting a book, registering a book and putting it up for rent, buying and selling books, user and book feedback systems, etc. For the admin, we included functionalities such as viewing feedback and taking action on necessary users/books, viewing customer queries, etc.
  • Wrote test benches for entire codebase using JEST framework with over 80% code coverage and used SWAGGER for auto documentation
  • The website was built using HTML, CSS, Bootstrap frontend and Javascript backend.

Deep Neural Networks

Domain: Neural Networks and Verilog Programming

  • Designed 8*8 and 16*16 fixed point approximate multipliers and Leading Zero Count in Verilog henceforth implementing it on FPGA.
